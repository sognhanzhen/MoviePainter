# MoviePainter 页面状态流转图

本文档描述 MoviePainter 当前阶段的页面状态流转与关键交互流程。

## 一、全局页面状态流转

```mermaid
flowchart TD
    A["未登录用户"] --> B["营销 landing 页"]
    B --> C["登录 / 注册"]
    C --> D["生成工作区页"]

    D --> E["海报库页"]
    D --> F["历史生成记录页"]
    D --> G["个人设置页"]
    D --> H["后台管理页"]

    E --> D
    F --> D
    G --> D
    H --> D
```

## 二、登录状态流转

```mermaid
stateDiagram-v2
    [*] --> 未登录
    未登录 --> 登录中: 提交登录/注册
    登录中 --> 已登录: 成功
    登录中 --> 未登录: 失败
    已登录 --> 未登录: 退出登录 / 登录态失效
```

## 三、海报库页状态流转

```mermaid
flowchart TD
    A["进入海报库页"] --> B["加载官方精选海报"]
    B --> C["展示海报列表"]
    C --> D["点击海报"]
    D --> E["打开海报详情弹窗"]
    E --> F["点击 Use"]
    F --> G["选择 AI Chat / AI Draw"]
    G --> H["进入生成工作区对应模式"]
```

## 四、生成工作区页状态流转

```mermaid
flowchart TD
    A["进入生成工作区页"] --> B["默认 AI Chat"]
    B --> C["用户切换模式"]
    C --> D["AI Chat 模式"]
    C --> E["AI Draw 模式"]

    D --> F["点击下方灵感海报"]
    E --> G["点击下方灵感海报"]

    F --> H["海报进入对话框上下文"]
    G --> I["海报进入 AI Draw 工作区展示"]
```

## 五、海报库进入工作区的状态流转

```mermaid
flowchart TD
    A["海报库页"] --> B["点击海报"]
    B --> C["海报详情弹窗"]
    C --> D["点击 Use"]
    D --> E["选择模式"]
    E --> F["AI Chat"]
    E --> G["AI Draw"]
    F --> H["进入工作区并带入海报"]
    G --> H
```

## 六、工作区内灵感海报使用流转

这是和海报库页不同的一条逻辑。

海报库页：

- 先看详情
- 再点 `Use`
- 再选模式

生成工作区页：

- 用户已经先处于当前模式
- 直接在下方灵感区选择海报
- 海报直接进入当前模式对应的工作区

```mermaid
flowchart TD
    A["当前位于生成工作区"] --> B["当前模式已确定"]
    B --> C["点击下方海报灵感区海报"]
    C --> D["识别当前模式"]
    D --> E["AI Chat: 注入对话框"]
    D --> F["AI Draw: 注入 Draw 工作区"]
```
